# agoraTokenServer

## Project setup

```
npm install
```

### Compile and serve token generator on port 8080

```
node index.js
```

### How to call it
```
http://localhost:8080/rtc/<channel_name>/publisher/uid/0
```
where <channel_name> will be replaced by some channel name

### Call the deployed API
